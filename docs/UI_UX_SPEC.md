# UI / UX Specification

## Main navigation

- Home
- Create
- Templates
- Prompt Library
- Scenario Library
- Community
- Competitions
- Pricing / Refill Tokens
- My Profile
- Admin Dashboard if admin

## Create flow

### Step 1 — Choose domain

Cards:

- Architecture
- Photography
- Branding

### Step 2 — Choose workflow/template

Examples:

Architecture:

- Exterior render
- Interior render
- Facade redesign
- Landscape concept
- Site plan enhancement
- Architectural clip
- Real estate marketing visual

Photography:

- Product hero shot
- E-commerce pack
- Luxury editorial
- Food photography
- Fashion campaign
- Product clip

Branding:

- Brand moodboard
- Logo direction
- Campaign visual
- Social media pack
- Real estate launch
- Brand clip

### Step 3 — Upload references

Every reference must have a role:

- Geometry reference
- Style reference
- Material reference
- Lighting reference
- Camera reference
- Brand reference
- Product reference
- Motion reference

### Step 4 — Smart dropdowns

Display only relevant dropdowns based on domain and template.

Each dropdown should show:

- Option label
- Best for
- English description
- Arabic description

### Step 5 — Prompt preview

Show:

- Main prompt
- Negative prompt
- Engine-specific prompt
- Estimated token cost
- Prompt length warning
- Quality warnings

### Step 6 — Generate

Actions:

- Generate draft
- Generate final
- Save prompt
- Save scenario
- Add to project

### Step 7 — Output result

Actions:

- Upscale
- Make clip
- Add audio
- Regenerate variation
- Fix issue
- Save to library
- Publish to community
- Submit to competition

## Modes

### Simple Mode

Minimal controls. Strong defaults.

### Advanced Mode

All professional controls visible.

### Expert Mode

Custom template/scenario creation.
